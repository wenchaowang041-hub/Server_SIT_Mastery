# BMC Cases

这个目录只保留后续经过整理、具备长期复用价值的 BMC / Redfish / IPMI 正式案例。

适合放在这里的内容：

- 多次复用的标准排障案例
- 已经抽象出方法论的告警分析
- 能沉淀进知识库的典型根因与处置流程

不适合直接放在这里的内容：

- 当天现场的原始工作记录
- 还未整理完成的临时排障笔记

建议命名方式：

- `YYYY-MM-DD-设备IP-主题.md`

建议每篇正式案例至少包含：

- 设备信息
- 告警汇总
- 关键日志
- 根因判断
- 处理建议
- 复查结论
