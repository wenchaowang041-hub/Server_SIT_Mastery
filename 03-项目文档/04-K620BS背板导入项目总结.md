## 5. NVMe 性能分析

### 5.1 分析架构

**位置**: `nvme分析脚本/all_nvme_analysis/`

覆盖 20 块 NVMe 盘: nvme1n1 ~ nvme19n1

**每盘分析 5 种测试模式**:
| 模式 | 说明 |
|------|------|
| 4K_Seq_read | 4K 顺序读 |
| 4K_Seq_write | 4K 顺序写 |
| 4K_RandRW_50read | 4K 随机混合读写 (50%读) |
| 128K_Seq_read | 128K 顺序读 |
| 128K_Seq_write | 128K 顺序写 |

**每个测试输出**:
- `result.json` - 原始JSON数据
- `summary.txt` - 测试总结 (PASS/FAIL)
- `plots/*.png` - 性能曲线图

### 5.2 分析示例 (nvme1n1)

```
采样点数: 2881
盘类型: SSD
跌零点数: 0 (阈值 1.0 MB/s)
稳态分析类型: SSD
整体波动点占比: 2/2881 = 0.07%
最终判定: PASS (128K_Seq_read)
```

**128K 顺序写**:
```
最终判定: FAIL
```

### 5.3 分析索引

每块盘都有 `index.txt` 汇总所有测试结果:
- 文件数量、分析状态、跌零数、稳态类型、PASS/FAIL
