**稳定性测试**（BMC界面中通电开机策略：保持上电）

Ubuntu系统中没有rc.loca，l先创建rc.local步骤：

cp /lib/systemd/system/rc-local.service /etc/systemd/system/（复制服务文件）

touch /etc/rc.local （创建rc.local脚本文件）

chmod 755 /etc/rc.local （设置文件权限）

echo '''#!/bin/bash''' \>\> /etc/rc.local （在rc.local文件开头添加shebang，指定使用bash shell来执行脚本）

sudo chmod a+x /etc/rc.local （再次确保可执行权限）

systemctl start rc-local （启动服务）

systemctl enable rc-local （启动 rc-local服务）开机自启

sudo systemctl daemon-reload （重新加载 systemctl 的配置文件，使更改生效）

sudo systemctl status rc-local （查看rc.local服务的运行状态）

rc-local.service增加内容( vi /etc/systemd/system/rc-local.service )

\[Install\]

WantedBy=multi-user.target

Alias=rc-local.servicem
